About

Theorem is a StudyPath Engine.

Theorem is designed to organize learning material before tutoring begins. The extraction layer reads pasted or uploaded text. Then the app’s code parses the material into folders like questions, key terms, dates, events, math problems, writing prompts, code blocks, sources, and unknown items.

Based on the material you provided, Theorem detects the subject, document type, task type, and suggested study method. Then it builds tools like flashcards, short-answer practice, timeline review, cause/effect practice, outline building, exact math checking, code review prompts, and a review plan.

This version is frontend-only and GitHub Pages-compatible: HTML, CSS, JavaScript, no backend, no paid API, no Google login, and localStorage only.
