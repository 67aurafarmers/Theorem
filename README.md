# Theorem

Theorem is a mistake-first Algebra 1 tutoring website.

It does not simply tell students the answer. It helps them find the step they missed, understand the mistake pattern, and practice the missing skill.

## Core promise

**Find the step you missed. Master the idea behind it.**

## Features

- Static website
- Works on GitHub Pages
- No backend
- No paid API
- No login
- No external dependencies
- Local progress saving with `localStorage`
- Diagnostic feedback for:
  - Two-step equations
  - Combining like terms
  - Distributive property
- Mistake pattern tracking
- Progress dashboard
- Review recommendations
